    Volver a un Commit Anterior

    Crear un repositorio en GitHub:

    • Inicia sesión en tu cuenta de GitHub.

    • Haz clic en el botón verde "New" (Nuevo) en la esquina superior derecha de tu página de inicio.

    • Asigna un nombre al repositorio, elige la visibilidad (público o privado) y selecciona la opción para inicializar el repositorio con un archivo README.md (esto es opcional, pero puede ser útil para empezar).

    • Haz clic en el botón "Create repository" (Crear repositorio).

    Clonar el repositorio en tu computadora:

    • Abre tu terminal (en macOS y Linux) o Git Bash (en Windows).



    • Navega hasta la ubicación donde quieres que se clone el repositorio usando el comando cd ruta/de/la/carpeta.

    • Clona el repositorio usando el comando git clone URL_del_repositorio. Puedes encontrar la URL del repositorio en la página principal del repositorio en GitHub.

    Agregar archivos al repositorio:

    • Coloca los archivos que deseas subir en la carpeta del repositorio clonado.

    • Abre tu terminal o Git Bash en la carpeta del repositorio clonado.

    Añadir cambios al área de preparación:

    • Usa el comando git status para verificar los cambios en tu repositorio local.

    • Para agregar todos los cambios, usa git add . o para agregar archivos específicos, usa git add nombre_del_archivo.

    Confirmar los cambios:

    • Usa el comando git commit -m "Mensaje de commit descriptivo" para confirmar los cambios en tu área de preparación. Sustituye "Mensaje de commit descriptivo" por un mensaje que describa los cambios realizados.

    Subir los cambios al repositorio remoto (GitHub):

    • Usa el comando git push origin main para subir los cambios a la rama principal del repositorio remoto en GitHub. Si estás utilizando una rama diferente, sustituye "main" por el nombre de tu rama.
